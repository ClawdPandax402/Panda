# Roadmap

Development priorities and upcoming features.

---

## Completed

### v1.0.0

- [x] Core x402 payment protocol
- [x] Solana mainnet support
- [x] Server-side gatekeeper middleware
- [x] Client SDK for browser and Node.js
- [x] Agent runtime with budget controls
- [x] Privacy mode
- [x] Domain whitelisting
- [x] Comprehensive documentation
